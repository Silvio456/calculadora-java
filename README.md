# 🧮 Calculadora em Java

Projeto de uma calculadora simples desenvolvida em Java, executada via terminal. Permite realizar operações matemáticas básicas entre dois números informados pelo usuário, com validação de entrada e tratamento de erros.

## ⚙️ Funcionalidades

- Soma
- Subtração
- Multiplicação
- Divisão (com validação para divisão por zero)
- Execução em loop até o usuário encerrar o programa
- Tratamento de entradas inválidas

## 💻 Tecnologias utilizadas

- Java
- Scanner (entrada de dados via console)

## 🎯 Objetivo

Este projeto foi desenvolvido com o objetivo de praticar lógica de programação em Java, explorando estruturas condicionais, laços de repetição e manipulação de entrada de dados pelo usuário.
